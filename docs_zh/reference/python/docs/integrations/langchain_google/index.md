# :material-google:{ .lg .middle } Google 集成

## 模块

!!! note "使用文档"
    关于如何使用每个模块的高级指南，请参阅[文档](https://docs.langchain.com/oss/python/integrations/providers/google)。这些参考页面包含每个模块自动生成的 API 文档，侧重于“是什么”而非“如何”或“为什么”（即不包含端到端教程或概念概述）。

LangChain 的 Google 集成包含以下包：

<div class="grid cards" markdown>

- :simple-googlegemini:{ .lg .middle } __Gemini__

    ---

    `langchain-google-genai` 包含与 Google Gemini API 交互的集成。

    [:octicons-arrow-right-24: 参考](../langchain_google_genai/index.md)

- :simple-googlecloud:{ .lg .middle } __Vertex AI 平台__

    ---

    `langchain-google-vertexai` 提供与 Google Cloud Vertex AI 平台协作的集成。

    [:octicons-arrow-right-24: 参考](../langchain_google_vertexai/index.md)

- :fontawesome-solid-people-group:{ .lg .middle } __社区__

    ---

    `langchain-google-community` 托管社区贡献的集成，涵盖未包含在 Google AI 产品中的各类 Google 服务。

    [:octicons-arrow-right-24: 参考](../langchain_google_community/index.md)

</div>
