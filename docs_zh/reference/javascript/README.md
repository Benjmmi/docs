# LangChain TypeScript 参考文档

此目录包含 TypeScript 参考文档站点的源代码和构建流程，该站点托管于 [`reference.langchain.com/javascript`](https://reference.langchain.com/javascript)。此站点提供 LangChain、LangGraph、LangSmith 以及 LangChain 集成包（例如 [`@langchain/anthropic`](https://npm.com/package/@langchain/anthropic)、[`@langchain/openai`](https://npm.com/package/@langchain/openai) 等）的参考文档。
